# CodyUI Pharmacy Theme Previews

This folder contains the design concepts and prompts for the marketplace previews of the CodyUI Pharmacy theme.

## How to Generate Previews
Use the following prompts with a high-quality AI image generator (like Midjourney, DALL-E 3, or Gemini) to create the final assets for the Salla Marketplace.

### 1. Homepage Preview (1920x1080)
**Prompt:** `A professional marketplace preview of a modern Arabic online pharmacy website homepage called 'CodyUI'. RTL layout. Clean white background with soft teal (#0D9488) and mint green accents. Hero section with a banner showing high-quality health products. Category grid with circular icons for medicine, vitamins, and baby care. Featured product grid with clean, rounded cards. Arabic typography (IBM Plex Sans Arabic). Salla platform style. High resolution, 1920x1080.`

### 2. Product Page Preview (1920x1080)
**Prompt:** `A professional screenshot of a product page for an online pharmacy. Arabic RTL. Large product image of a vitamin bottle. Price and 'Add to Cart' button in bold teal. Prescription required badge in soft red. Medicine information tabs (Description, Usage, Warnings) visible. Related products at the bottom. Clean, clinical design with soft shadows. 1920x1080.`

### 3. Category Page Preview (1920x1080)
**Prompt:** `A professional screenshot of a category listing page for an online pharmacy. Arabic RTL. Sidebar with filters for brand, price, and availability. Grid of medical products with 'Authentic' and 'Sale' badges. Clean white UI, soft shadows, teal accents. Arabic text. 1920x1080.`

### 4. Mobile Homepage Preview (1080x1920)
**Prompt:** `A professional mobile screenshot of an online pharmacy homepage. Arabic RTL. Sticky header with search bar. Hero banner. Horizontal scrolling categories. Vertical product list. Modern, rounded UI with touch-friendly buttons. 1080x1920 aspect ratio.`

### 5. Customization Preview (1920x1080)
**Prompt:** `A professional UI design showing the theme customization settings for CodyUI Pharmacy theme on Salla. Color palette selection (Teal, Mint, White). Typography settings (Arabic fonts). Toggle switches for pharmacy services like 'Prescription Upload'. Clean dashboard interface with a live preview of the site. 1920x1080.`

### 6. Marketplace Cover (1600x900)
**Prompt:** `A high-quality marketplace cover for 'CodyUI – Pharmacy E-commerce Theme for Salla'. Text: 'CodyUI' in bold teal. Subtitle: 'Modern Arabic RTL theme for online pharmacies'. Background features a blurred medical store interior with clean UI elements floating. Professional, trustworthy aesthetic. 16:9 aspect ratio (1600x900).`
